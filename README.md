# Bonsai - a Ruby on Rails example app

This Ruby on Rails app is used during Ruy on Rails Advanced training.

## Requirements

- Ruby 2.5+
- SQLite3
- NodeJS
- Yarn

## Installation

```
bundle install
yarn install
rails db:setup
```

## Usage

```
rails s
```

Then go to [localhost:3000](http://localhost:3000)
